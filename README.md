# Treeview

This is a gemefied version of Ordie's treeview.

## Installation

Add this line to your application's Gemfile:

    gem 'rails-treeview'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rails-treeview

## Usage

- Add the module "tree-view" to your AngularJS app
